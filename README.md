# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/folfer/desafio-albert
```

```bash
# Install Dependencies
$ yarn

# Run Aplication
$ yarn ios or yarn android

# Run Test
$ yarn test
```
